Agentic Job Application Pipeline

Built a guarded multi-agent flow that sourced roles, tailored resumes, drafted outreach, and tracked progress with human approvals.

Role: Product + delivery Timeline: 6 weeks Stack: GPT-4o, vector search, Notion
AI Product Automation
Work together Back to work

Situation

Job searches were manual and inconsistent. Users needed automation without risky unsupervised outreach.

  • Multiple job boards to monitor daily
  • Manual resume tailoring and follow-up emails
  • No consolidated view of which outreach step was next

Approach

Designed an agentic pipeline with hard stops for human approval.

  • Agent 1: Source roles via APIs + RSS, dedupe, and score against preferences
  • Agent 2: Tailor resume + cover letter using a controlled prompt library
  • Agent 3: Draft outreach and schedule follow-ups; user approves before send
  • Simple dashboard: queue, approvals, and metrics

Safety & Governance

  • Guardrails for tone, length, and company-specific facts
  • Deterministic prompt templates checked into version control
  • Event logging for every agent step
  • Manual override at each outbound action

Outcomes

  • Daily sourcing dropped from ~45 minutes to under 10
  • Outreach accuracy improved because approvals were structured
  • Users could see exactly what was in-flight and what needed review
